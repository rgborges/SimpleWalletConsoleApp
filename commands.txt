#Commands

Sellect a Wallet instance into the program:

    SimpleWallet>--select-wallet                            | Selects a wallet using its guid using wizard process.

    SimpleWallet>--select-wallet -w <guid>                  | Selects a wallet using its guid.

Adds a receive:

    SimpleWallet>--add-receive

Add a spending:

    SimpleWallet>--add-spending

Add a wallet to the system

    SimpleWallet>--add-wallet

Show info

    SimpleWallet>--show-info
